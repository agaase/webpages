Genwi.Plugins.Superscroll Class

Defined in: src/plugins/incscroll/Superscroll.js:4
Module: Genwi

This is a simple incremental scroller which uses the normal window scroll to trigger an action when the user reaches bottom of the page.

superScroll

(
  • options
)

Defined in src/plugins/incscroll/Superscroll.js:141

Call this function to initialise scrolling. Following options are to be set 1.totalRecords:(optional,default-100) Total number of records present. 2.maxRecords: (optional,default 100) Max number of records to be fetched. 2.setcount: (optional,default 20) the number of items that will be fetched on next fetch. 3.recordsLoaded: (required) Records already loaded. 4.callbackFn: (optional, default implementation present) The function to be executed when the scroll reaches bottom of the container. 5.loadingMessage: (optional-default is the css loader) loading message to display

Parameters:

  • options Object
    • Object containing the configuration options
