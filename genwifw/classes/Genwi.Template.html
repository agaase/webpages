Genwi.Template Class

Defined in: src/Templates.js:25
Module: Genwi

This whole thing is essentially a very stripped down Backbone.js, since our Router and to a large extent Model are already taken care of by native. All it does is take Backbone's subclassing and adds in an abstract class that can be used to build better templates.
